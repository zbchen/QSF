*smtlib_qf_fp.tar.gz* and *program_qf_fp.tar.gz* are two sets of evaluation benchmarks in the paper, which are from SMTLIB-Comp and real-world programs respectively.

Run `cat smtlib_qf_fp.tar.gz_* > smtlib_qf_fp.tar.gz` in this path to get *smtlib_qf_fp.tar.gz*.

Run `cat program_qf_fp.tar.gz_* > program_qf_fp.tar.gz` in this path to get *program_qf_fp.tar.gz*.
